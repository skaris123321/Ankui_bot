# Как сделать чтобы бот не засыпал локально

## Вариант 1: Настройки Windows

### Отключить спящий режим:
1. Пуск → Параметры → Система → Питание и спящий режим
2. Установите "Никогда" для всех пунктов
3. Дополнительные параметры питания → Изменить параметры схемы
4. Переводить компьютер в спящий режим: Никогда

### Запуск бота как службы Windows:
1. Установите pm2 глобально:
```cmd
npm install -g pm2
npm install -g pm2-windows-service
```

2. Настройте службу:
```cmd
pm2-service-install
pm2 start index.js --name "discord-bot"
pm2 save
```

## Вариант 2: Использовать VPS/хостинг (рекомендуется)

### Бесплатные варианты:
- **Railway** (рекомендуется) - 500 часов в месяц бесплатно
- **Render** - засыпает через 15 минут неактивности
- **Heroku** - больше не бесплатный
- **Glitch** - засыпает через 5 минут

### Платные варианты (дешевые):
- **VPS от Timeweb** - от 100₽/месяц
- **VPS от Beget** - от 150₽/месяц
- **DigitalOcean** - от $5/месяц
