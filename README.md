# 🤖 Discord Bot с Веб-Панелью Управления

Многофункциональный Discord бот с удобной веб-панелью управления, похожий на ProBot, FlaviBot и FA Rooms.

## 📋 Требования

- Node.js 16.9.0 или выше
- npm или yarn
- Discord Bot Token

## 🛠️ Установка

### 1. Установите зависимости

```bash
npm install
```

### 2. Настройте .env файл

Откройте файл `.env` и настройте следующие параметры:

```env
# ОБЯЗАТЕЛЬНО замените токен на новый после сброса!
DISCORD_TOKEN=ваш_новый_токен_здесь
CLIENT_ID=1443654730239709206
GUILD_ID=id_вашего_сервера (опционально)

# Настройки веб-панели
PORT=3000
SESSION_SECRET=случайная_строка_для_безопасности

# OAuth2 (для будущей авторизации через Discord)
CLIENT_SECRET=ваш_client_secret_здесь
CALLBACK_URL=http://localhost:3000/auth/callback

# Twitch API (ОПЦИОНАЛЬНО - для более надежной работы уведомлений о стримах)
# Получить можно на https://dev.twitch.tv/console
# 1. Зарегистрируйтесь на https://dev.twitch.tv
# 2. Нажмите "Register Your Application"
# 3. Заполните форму (Name: любое, OAuth Redirect URLs: http://localhost)
# 4. Скопируйте Client ID
TWITCH_CLIENT_ID=ваш_twitch_client_id_здесь

# YouTube API (ОПЦИОНАЛЬНО - для уведомлений о стримах YouTube)
# Получить можно на https://console.cloud.google.com/
# 1. Создайте проект в Google Cloud Console
# 2. Включите YouTube Data API v3
# 3. Создайте API ключ
# 4. Скопируйте ключ
YOUTUBE_API_KEY=ваш_youtube_api_key_здесь
```

### 3. Зарегистрируйте slash-команды

```bash
node deploy-commands.js
```

### 4. Запустите бота

```bash
npm start
```

Или для разработки с автоперезагрузкой:

```bash
npm run dev
```

## 🌐 Веб-панель управления

После запуска откройте браузер и перейдите на:

```
http://localhost:3000
```

### Основные страницы:

- `/` - Главная страница
- `/dashboard` - Панель управления
- `/guild/:id` - Настройки конкретного сервера
- `/docs` - Документация

## 🎮 Команды бота

### Основные команды

- `/help` - Показать список всех команд
- `/ping` - Проверить задержку бота
- `/userinfo [пользователь]` - Информация о пользователе
- `/serverinfo` - Информация о сервере

### Модерация (требуются соответствующие права)

- `/clear [количество]` - Удалить сообщения (1-100)
- `/kick [участник] [причина]` - Кикнуть участника
- `/ban [участник] [причина]` - Забанить участника
- `/mute [участник] [минуты] [причина]` - Замутить участника
- `/warn [участник] [причина]` - Выдать предупреждение

## ⚙️ Настройка через веб-панель

### Приветствия

1. Включите систему приветствий
2. Укажите ID канала для приветствий
3. Настройте сообщение (доступны переменные: `{user}`, `{username}`, `{server}`, `{membercount}`)

### Прощания

1. Включите систему прощаний
2. Укажите ID канала для прощаний
3. Настройте сообщение прощания

### Автороль

Укажите ID роли, которая будет автоматически выдаваться новым участникам

## 🔧 Как получить необходимые ID

### ID сервера (Guild ID)

1. Включите режим разработчика в Discord: Настройки → Расширенные → Режим разработчика
2. Нажмите правой кнопкой на название сервера
3. Выберите "Копировать ID сервера"

### ID канала (Channel ID)

1. Включите режим разработчика (как выше)
2. Нажмите правой кнопкой на канал
3. Выберите "Копировать ID канала"

### ID роли (Role ID)

1. Откройте настройки сервера → Роли
2. Нажмите правой кнопкой на роль
3. Выберите "Копировать ID роли"

## 🔐 Пригласить бота на сервер

1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Выберите ваше приложение
3. Перейдите в раздел OAuth2 → URL Generator
4. Выберите scopes: `bot` и `applications.commands`
5. Выберите необходимые права:
   - Manage Roles (Управление ролями)
   - Kick Members (Кик участников)
   - Ban Members (Бан участников)
   - Moderate Members (Модерация участников)
   - Manage Messages (Управление сообщениями)
   - Read Messages/View Channels (Чтение сообщений)
   - Send Messages (Отправка сообщений)
   - Embed Links (Встраивание ссылок)
   - Attach Files (Прикрепление файлов)
   - Read Message History (Чтение истории)
6. Скопируйте сгенерированную ссылку и откройте в браузере

## 🐛 Решение проблем

### Бот не запускается

- Проверьте, что установлены все зависимости: `npm install`
- Проверьте правильность токена в `.env` файле
- Убедитесь, что используете Node.js 16.9.0 или выше

### Команды не отображаются

- Запустите `node deploy-commands.js` для регистрации команд
- Подождите до 1 часа для глобальной регистрации команд
- Убедитесь, что у бота есть права на использование slash-команд

### Веб-панель не открывается

- Проверьте, что порт 3000 не занят другим приложением
- Измените PORT в `.env` файле на другой
- Проверьте firewall/антивирус
